## Installation

### Composer

Execute the following command to get the latest version of the package:

```terminal
composer require dovutuan/laralog
```

### Publish Configuration

```shell
php artisan vendor:publish --tag=laralog
```

### Config env

```dotenv
APP_LOG_QUERY=true
```
